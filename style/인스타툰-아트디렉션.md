# 인스타툰 아트디렉션 · 마스터 문서

이 문서는 **[[이두형]] 가족 인스타툰**의 모든 시각 결과물이 따라야 할 **연출 바이블**이다.
캐릭터 컨셉 시트·스토리보드 컷·즉석 이미지 — **모든 이미지 생성**은 이 문서의 규칙을 절대적으로 따른다.

> ⚠ 이 볼트의 아트 관련 판단이 충돌할 때, 이 문서가 최상위 기준이다.
> 톤을 조정해야 하면 **이 문서 한 곳만** 고친다. 각 캐릭터 카드는 이 문서를 참조만 한다.

> 🔗 **연출 톤(잔잔 vs 코믹) 의 비율과 비트 분할은 [[knowledge/인스타툰-코믹-연출|코믹 연출 마스터]] 가 정의한다.** 이 문서는 \"잔잔의 디폴트 비주얼\" 을, 코믹 마스터는 \"한 화 안에서 어디서 일탈할지\" 를 다룬다. 두 문서는 항상 같이 적용.

## 🎯 한 줄 정의

> **\"그림책 한 페이지 같은, 플랫하고 따뜻한 일상 인스타툰 — 단, 화당 1~2컷은 빵 터지는 코믹 일탈을 허용한다.\"**
>
> 웹툰 아님. 영화 아님. 일러스트 에세이 + 스파이스 한 꼬집.

### 레퍼런스 감각
- 이말년 느낌 ❌ · 조석 느낌 ❌ · 극화 웹툰 ❌
- 감성 인스타툰 / 에세이 일러스트 / 그림책 / 리소그래프 포스터 ✅
- 키워드 (잔잔 디폴트): **잔잔함 · 여백 · 플랫 · 따뜻 · 손맛 · 생활감**
- 키워드 (코믹 일탈): **격차 · 진폭 · 무방비 · 영혼가출 · 빵**

## ✅ 필수 규칙 (Must) — 잔잔 디폴트

### 1. 라인 (Line)
- **얇고 손맛 있는 펜선**. 굵기가 일정하지 않고 끊어질 듯 이어지는 가벼운 터치.
- 컨투어 컬러는 **따뜻한 브라운-블랙 `#3a2e28`**.
- 선 끝은 살짝 가늘어지며 자연스럽게 떨어짐.
- **코믹 컷 예외**: 펀치라인 컷에 한해 컷 테두리에 **얇은 흔들림 효과선** 또는 **방사형 가는 폭발선** 1세트 허용 (아래 §9 참조).

### 2. 채색 (Color Fill)
- **완전 플랫 컬러 필**.
- 셀셰이딩은 **최대 1단**, 투명도 **15~20%**.
- 피부 홍조는 뺨에 **옅은 핑크 원 하나**만.

### 3. 배경 (Background) — 🔥 가장 중요
- 원칙: **\"필수 정보만\"**. 사무실 집기·TV 화면·창밖 야경 등은 **대담하게 생략**.
- 베이스 컬러:
  - **아이보리 `#f4efe6`** (기본)
  - **연크림 `#f7f2ea`** (따뜻한 장면)
  - **연회색 `#e8e8ec`** (차가운 장면)
- 공간감은 **단색 면 + 소품 1~2개**로만.
- 조명 변화는 **배경 색 한 톤 바꾸기**로만. 광원·플레어·빛줄기 금지.
- **코믹 펀치라인 컷 예외**: 배경을 일시적으로 **단색 + 방사형 폭발선** 으로 처리해 표정만 부각하는 연출 1컷 허용 (예: 깨달음·외침·영혼가출의 백색 배경).

### 4. 구도 (Composition)
- **정면성·평면성** 위주. 납작한 정면/측면/살짝 비스듬.
- **인물이 캔버스의 60% 이상**, 나머지는 **의도적 여백**.
- 말풍선이 레이아웃의 일부.
- **코믹 펀치라인 컷 예외**: extreme-close (얼굴만 화면 가득) 또는 살짝 비뚤어진 앵글 허용.

### 5. 레터박스 (Letterbox)
- 모든 컷 위·아래 **얇은 가로 여백 띠** (좌우 없음).
- 두께: 세로 높이의 **5~7%**.
- 컬러: **크림 아이보리 `#ece5d8`**.
- 경계: 라인·그림자 없음.
- 내용 침범 금지.
- 캐릭터 컨셉 시트(턴어라운드) 에는 적용하지 않음.

### 6. 이펙트 & SFX — **모드 분리 필수**
이 섹션은 [[knowledge/인스타툰-코믹-연출]] §3 과 한 쌍으로 작동한다.

#### 6a. **잔잔 모드** (전체 컷의 70~80%, 디폴트)
- 하트·반짝임·땀방울·물음표 — **작고 가늘게**, 수량 최소.
- SFX 손글씨는 **삐뚤빼뚤 얇게**, 손글씨 느낌.
- 이펙트 동시 적용 금지 (땀+하트 같이 X).

#### 6b. **코믹 모드** (펀치라인 컷에만, 화당 1~2컷)
펀치라인 컷에서는 아래 만화 문법을 **그 컷 1개에 한해** 허용한다:

| 이펙트 | 강도 |
|---|---|
| 점선 눈동자 (`•_•`) | 자유 |
| 만화적 땀줄기 (얼굴 옆 큰 물방울 1개) | 1개만 |
| 이마 핏줄 (`#`) | 1~2개 |
| 점선 동공 / 무광 눈 | 자유 |
| 입 ㅁ자 / 입 ω자 | 펀치라인 1컷 |
| 머리 위 뿅뿅 (작은 별 2~3개) | 자유 |
| 만화적 음표·하트 큼지막하게 | 1개만 크게 |
| 컷 흔들림 효과선 | 펀치라인 1컷 |
| 백색 배경 폭발선 | 펀치라인 전용 |

**코믹 모드 금지 사항** (잔잔 모드와 공통):
- 굵은 검은 외곽선·고대비 잉킹은 여전히 금지.
- 그라데이션·광원·플레어 여전히 금지.
- **이펙트 다중 적용 금지** — 코믹 모드라도 한 컷에 1~2종까지. 땀+핏줄+별 동시는 만화 데뷔작 톤이 됨.
- 선 두께·플랫 컬러 룰은 그대로. 표정과 이펙트만 일탈.

> 💡 핵심: 코믹 모드는 **\"잔잔 톤 안에서 한 컷만 일탈\"** 이라 웃기다. 매 컷 코믹 = 피로함.

### 7. 컬러 팔레트
- **따뜻한 어스톤** 중심: 브라운 · 올리브 · 베이지 · 크림.
- 포인트 컬러:
  - [[오은주]]: **검정 + 데님 블루**
  - [[이두형]]: **브라운/차콜 + 올리브/카키**
  - [[이재윤]] (밤톨이): **베이지/아이보리**
  - [[이채은]] (곶감이): **분홍/연노랑/크림 파스텔**
- 채도 낮게, 명도 높게.

### 8. 이미지 내 텍스트 — 🆕 절대 금지
- **이미지 안에 어떠한 글자도 렌더링하지 않는다.**
- 말풍선의 모양은 그릴 수 있지만 **풍선 안은 비워둔다**.
- **내레이션 박스도 비워두거나 아예 그리지 않는다**.
- 효과음(SFX)은 시각적 형태 — 깨진 선·진동 — 으로만 암시. 글자 X.
- 간판·메뉴판·핸드폰·TV·책 표지 등 **배경 모든 텍스트는 빈 면 또는 추상 무늬**.
- (속으로) (한숨) 같은 **괄호 지문이 글자로 새겨지는 사고 방지**.
- 단, **캐릭터 턴어라운드 시트의 라벨**(Front/Side/Back) 은 예외.

### 9. 표정 진폭 (NEW) — 코믹 마스터와 한 쌍
[[knowledge/인스타툰-코믹-연출]] §1 의 **표정 5단계** 를 시각 규정으로 흡수한다:

| 단계 | 표정 비주얼 | 그림 처리 |
|---|---|---|
| L1 잔잔 | 반달눈, 옅은 미소 | 디폴트, 전체의 50~60% |
| L2 시큰둥/의심 | 한쪽 눈썹↑, 입꼬리 한쪽 | 평소 라인 그대로 |
| L3 당황/놀람 | 눈 동그랗게, 입 작게 ㅇ | 평소 라인 그대로 |
| **L4 영혼가출** | 점선 눈동자, 입 일자 | **코믹 모드 이펙트 1종 허용** |
| **L5 폭주/절창** | 입 크게 ㅁ자, 눈 X자/별 | **코믹 모드 이펙트 1~2종 + extreme-close** |

> ⚠ **L4 또는 L5 표정 컷이 화당 최소 1개** 들어가야 한다. 안 들어가면 시리즈 톤이 평평해진다.

| 항목 | 금지 | 대체 |

| 채색 | 광택·반사·그라데이션·하이라이트 | 완전 플랫 컬러 필 |
| 배경 | 사진 같은 공간 묘사, 집기 풀세트 | 단색 면 + 소품 1~2개 (펀치라인 한정 폭발선 허용) |

| 레터박스 | 검정·짙은 회색 시네마 바, 높이 10% 초과 | 크림 아이보리 띠, 5~7% |
| 구도 | 영화적 부감·앙각 과용 | 정면성·평면성 (펀치라인 한정 extreme-close 허용) |
| **잔잔 컷의 코믹 이펙트** | **L1~L3 컷에 땀줄기·핏줄·점선동공 사용** | **펀치라인(L4/L5) 컷에서만** |
| **모든 컷이 잔잔 (L1~L2)** | **화당 L4/L5 가 0개** | **반드시 1개 이상 — 코믹 마스터 §1** |
| 이펙트 다중 사용 | 한 컷에 땀+핏줄+별 다 넣기 | 펀치라인이라도 1~2종까지 |
| **이미지 내 텍스트** | **말풍선/간판/SFX 글자** | **풍선 모양만, 안은 비움** |
| **본편 호칭** | **실명 일체** | **자기야 · 여보야 · 밤톨이 · 곶감이 4종만** |
| 키워드 | 웹툰, 시네마틱, 드라마틱 | 그림책, 잔잔한 에세이 + 코믹 일탈 |

## 🎬 컷 단위 체크리스트

스토리보드 컷을 생성하기 전, 각 panel 에 대해 아래를 확인한다.

### 모든 컷 공통
- [ ] 배경에 불필요한 디테일이 없는가?
- [ ] 조명을 광원이 아니라 배경 색으로 표현했는가?
- [ ] 인물이 캔버스의 60% 이상인가?
- [ ] 투샷이면 키 차이 22cm 가 드러나는가?
- [ ] 캐릭터별 포인트 컬러가 지켜졌는가?
- [ ] 상·하 레터박스 띠(크림 아이보리, 5~7%)가 들어갔는가?
- [ ] **이미지 안에 글자가 하나도 없는가?**
- [ ] **dialogue/narration 에 실명이 안 들어갔는가?**

### 잔잔 컷 (L1~L3)
- [ ] 이펙트가 \"작고 소량\" 인가? (코믹 모드 이펙트 사용 X)
- [ ] 구도가 납작한 정면성에 가까운가?

### 펀치라인 컷 (L4 또는 L5)
- [ ] 표정이 L4(영혼가출) 또는 L5(폭주/절창) 인가?
- [ ] 코믹 모드 이펙트가 **1~2종** 들어갔는가? (다중 사용 금지)
- [ ] 단독 컷인가? (subCuts 사용 금지)
- [ ] 앞 컷에 spacerAfter:\"beat\" 또는 \"silence\" 가 붙었는가? — [[knowledge/인스타툰-코믹-연출]] §2

### 화 단위
- [ ] L4 또는 L5 표정 컷이 **최소 1개** 있는가?
- [ ] 펀치라인 컷이 **1~2개** 있는가?
- [ ] 잔잔 컷 비율이 **50% 이상** 인가? (코믹 남발 금지)

## 🔧 프롬프트 공식 (이미지 생성 시)

이미지 생성 도구에 넣을 프롬프트에는 **항상 다음 문구를 머리에 포함**한다:

```
Style: flat illustrated Korean daily-life webtoon (일상 인스타툰), picture-book aesthetic.
Thin hand-drawn pen line in warm brown-black (#3a2e28), NOT pure black, NOT thick webtoon contour.
Completely FLAT color fill, at most 1 level of very soft cel-shading (15-20% opacity).
NO gloss, NO highlights, NO gradients, NO lens flare, NO dramatic lighting.
Background: solid cream/ivory (#f4efe6) with minimal props only — no detailed rooms,
no TV screens, no window scenery. Lighting mood conveyed ONLY through background color shift.
Composition: flat, frontal, character occupies 60%+ of canvas, generous negative space.
Subtle letterbox framing: thin horizontal bars on TOP and BOTTOM only (each ~5-7% of image height),
filled with warm cream-ivory (#ece5d8), half a tone darker than background. NO black cinema bars.
No line between letterbox and scene. Characters, bubbles, SFX stay INSIDE the inner frame.
Effects (hearts/sparkles/sweat) tiny and sparse for normal panels. SFX in thin wobbly handwritten style.
Warm earth-tone palette. Mood: calm, cozy, quiet essay-illustration.
NO text inside image (empty bubbles, no signage text, no SFX letters).
AVOID: cinematic, dramatic, high contrast, detailed backgrounds, webtoon inking, thick black cinema bars.
```

### 펀치라인 컷 전용 추가 문구 (L4/L5 표정 컷에만)

펀치라인 컷에는 위 베이스 프롬프트 뒤에 **다음 코믹 모드 블록을 추가**한다:

```
COMIC MODE PANEL — exaggerated expression allowed for THIS panel only:
- Character expression: [L4: vacant dotted-line eyes, flat horizontal mouth, soul-gone empty face]
  OR [L5: huge open round/square mouth, X-shaped or sparkling-star eyes, manga shouting face]
- Allowed manga effects (use 1~2 max, NOT all): single large sweat drop on forehead,
  one or two anger veins (#-shape), tiny stars/sparkles around head, oversized music note
  or oversized heart, faint motion-blur lines around character outline,
  white radial burst lines as background replacement.
- Composition: extreme close-up on face filling the frame, OR slightly tilted dutch angle.
- STILL FORBIDDEN even in comic mode: thick black inking, gradients, glow, lens flare,
  multiple effects stacked together, photo-realistic detail, text inside image.
- Line weight, flat fill, palette, letterbox — all UNCHANGED from base style.
```

> 💡 펀치라인 프롬프트는 \"추가\" 형태. 베이스 톤(라인·플랫·레터박스)은 그대로 유지하고 **표정과 1~2개 이펙트만** 일탈시킨다.
