# 인스타툰 코믹 연출 — 마스터 룰

이 작품의 톤은 **\"잔잔한 에세이 + 빵 터지는 코믹\"** 의 7:3 비율이다.
[[style/인스타툰-아트디렉션|아트디렉션 마스터]] 가 \"잔잔의 디폴트\" 를 정의한다면, 이 문서는 **\"코믹이 들어올 자리\"** 를 강제한다.

> ⚠ 이 룰이 없으면 작품은 \"따뜻하지만 평평한 일러스트 에세이\" 가 된다. 모든 화 집필 시 아래 체크리스트를 반드시 통과시킬 것.

---

## 🎯 진단 (왜 코믹이 안 살았나)

기존 1~6화의 공통 문제:

1. **표정 폭이 좁다** — \"옅은 미소·반달눈·살짝 곤란\" 3종 안에서만 표정이 움직임. 진폭이 없으니 코믹이 들어올 데가 없음.
2. **펀치라인 컷의 부재** — 화당 \"한 번은 빵 터져야 할\" 컷이 여전히 점잖게 그려져 있음. 콘셉트만 코믹, 그림은 잔잔.
3. **한 컷에 4대사 압축** — 비트가 시간차 없이 동시에 와서 개그 호흡이 죽음.
4. **시그니처 바보짓 부족** — 두형이 깐죽 / 은주 ESTJ 팩폭 / 쌍둥이 카오스가 캐릭터 카드엔 있는데 컷에선 안 보임.
5. **균일한 1:1 풀샷** — 클로즈업·인서트가 없어 시각 리듬이 평평.

## ✅ 절대 규칙

### 1. **표정 진폭 5단계** — 한 화에 최소 3단계 등장

| 단계 | 표정 | 사용 빈도 | 예시 |
|---|---|---|---|
| **L1 잔잔** | 반달눈, 옅은 미소, 디폴트 | 50~60% | 1화 6컷 \"이 맛에 산다\" |
| **L2 의심/시큰둥** | 한쪽 눈썹, 입꼬리 한쪽 | 15~20% | 은주의 \"또?\" |
| **L3 당황/놀람** | 눈 동그랗게, 입 작게 ㅇ | 10~15% | 두형이 들킨 순간 |
| **L4 영혼가출** | 점선 눈동자(•_•), 입 일자, 영혼 빠진 표정 | 5~10% | 잠 못 잔 새벽 두형, 쌍둥이 동시 울음 폭격 |
| **L5 폭주/절창** | 입 크게 벌림 ㅁ자, 눈 X자 또는 별, 만화적 과장 | 5% (펀치라인 컷 전용) | 4화 코노 컷, 은주 정색 폭격 |

> 📐 **L4·L5 는 화당 최소 1컷 의무**. 안 들어가면 그 화는 잔잔 톤에서 못 빠져나옴.
> L4 와 L5 는 같은 컷에서 같이 쓰지 말 것 (한쪽이 죽음). 다른 컷에 분리.

### 2. **펀치라인 컷 (Punchline Panel) — 화당 1~2개 의무**

펀치라인 컷의 정의:
- **표정이 L4 또는 L5**
- **만화적 이펙트 허용** (아래 \"코믹 모드 이펙트\" 참조)
- **단독 컷** (subCuts 사용 금지, 한 박자 단독으로 던질 것)
- **앞 컷에 spacerAfter:\"beat\" 또는 \"silence\"** — 비트 한 박자 쉬고 펀치가 들어가야 웃김

**펀치라인 위치 가이드**:
- 6~10컷 화: 4~5번째 자리 (한 번), 마지막 직전 (선택)
- 절대 금지: 1번 컷 펀치라인. 도입은 잔잔하게 깔고 들어가야 격차로 웃김.

### 3. **\"코믹 모드\" 이펙트 화이트리스트**

평소엔 [[style/인스타툰-아트디렉션]] 의 \"이펙트 작고 가늘게\" 를 따르지만, **펀치라인 컷에 한해** 아래 만화 문법을 허용한다:

| 이펙트 | 사용 상황 | 강도 |
|---|---|---|
| **점선 눈동자 (•_•)** | 영혼가출, 멍함, 한계 도달 | 자유 |
| **만화적 땀줄기** (얼굴 옆 큰 물방울 1개) | 당황, 식은땀 | 1개만, 머리 옆 |
| **이마 핏줄 (#)** | 분노 직전, 참고 있음 | 1~2개 |
| **점선 동공 / 무광 눈** | 영혼 빠짐, 자포자기 | 자유 |
| **입 ㅁ자 / 입 ω자** | 외침, 절창, 충격 | 펀치라인 컷 |
| **머리 위 뿅뿅** (당황 별 작은 거) | 놀람의 가벼운 버전 | 2~3개 |
| **만화적 음표·하트 큼지막하게** | 노래·애정 폭주 | 1개만 크게 |
| **컷 흔들림 효과선** (테두리 살짝 거친 선) | 충격·외침 컷 | 펀치라인 1컷 |
| **백색 배경 폭발선** (방사형 가는 선) | 깨달음·폭주 | 펀치라인 전용 |

여전히 **금지**:
- 굵은 검은 외곽선·고대비 잉킹
- 그라데이션·광원·플레어
- 이펙트 다중 적용 (땀+핏줄+별 동시 = 과함)
- 일반 잔잔 컷에 위 이펙트 사용

> 핵심: \"잔잔 톤 안에서 한 컷만 일탈\" 이라 웃김. 매 컷 코믹이면 피로함.

### 4. **비트 분리 — 한 컷 1~2 대사 원칙**

기존 잘못된 분할 (1화 2컷 사례):
> [한 컷에 4대사: 은주 \"어 아빠 오셨다!\" / 밤톨 \"우아—!\" / 곶감 \"으아아—!\" / 두형 속마음]

올바른 코믹 분할:
> [컷A: 은주가 두형 보고 \"어, 아빠 오셨다—\" + 쌍둥이 정적] →
> [컷B (subCuts 2): 밤톨/곶감 동시에 입 크게 벌림 → 동시에 \"우아아—!\"] →
> [컷C: 두형이 영혼가출 표정 + 속마음 \"...왔구나, 나.\"]

핵심:
- **한 컷 dialogue 는 1~2줄**. 3줄 이상이면 분할 검토.
- **반응샷은 별도 컷** — 누가 뭘 했으면 → 누가 어떤 표정으로 받는지 다음 컷에서.
- **속마음(thought) 과 외부 대사(speech) 가 같이 있으면** 거의 항상 분할.

### 5. **6컷 도그마 깨기 — 8~12컷 권장**

인스타 1포스트 최대 10장. 12장이 필요하면 캐러셀 2개로 가도 됨.

- **잔잔 톤 일상툰** (큰 사건 없음): 6~8컷
- **코믹 위주 화** (펀치라인 2개 이상): 8~10컷
- **이벤트 화** (시즌·기념일·큰 결정): 10~12컷

펀치라인 컷이 들어가려면 그 앞에 \"비트(beat) 컷\" 이 1~2개 있어야 격차가 살아남. 6컷에 펀치라인까지 넣으려고 하면 도입이 잘려서 톤 빌드업이 안 됨.

### 6. **시각 리듬 — 사이즈/샷 다양화**

한 화 안에서:
- **establishing wide** 1컷 (도입)
- **medium / bust** 다수 (대화 비중)
- **close / extreme-close** 1~2컷 (감정 고조 / 펀치라인)
- **insert** 1컷 (소품·손 클로즈업, 분위기 환기)
- **bird (살짝 부감)** 0~1컷 (음식·체육관 매트 등)

같은 샷이 3컷 이상 연속되지 말 것. **펀치라인 컷은 close 또는 extreme-close 가 디폴트** — 표정이 화면을 꽉 채워야 웃김.

## 🎭 캐릭터별 코믹 비트 카드

집필 시 \"이 화에 이 트리오 비트가 들어갔나?\" 자체 검수.

### [[이두형]] — \"깐죽 / 능청 / 영혼가출\"
- **시그니처 깐죽**: 은주가 정색하면 \"…응?\" 하고 모르는 척, 또는 \"아니 나는—\" 으로 변명 시작
- **능청 비트**: 사진부터 찍고 \"먹는 건 그 다음.\" 같은 본인만의 우주 룰
- **영혼가출 (L4)**: 잠 못 잔 새벽, 쌍둥이 동시 울음, 은주 정색 직격
- **절창 (L5)**: 코노에서, 또는 좋아하는 음식 앞에서 혼자 감격
- **금지**: 두형이를 너무 무력하게만 그리지 말 것. 작은 능청 한 줄이 항상 들어가야 캐릭터가 살아남.

### [[오은주]] — \"ESTJ 팩폭 / 시큰둥 / 의외의 폭소\"
- **팩폭 비트**: 두형이 변명 시작 → 은주 한 마디로 종결 (\"이번 주에만 세 번째.\")
- **시큰둥 (L2)**: 한쪽 눈썹, 입꼬리 한쪽. \"또?\" \"…먹어 그럼\" 디폴트
- **빵 터지기 (L5)**: 본인이 시큰둥하다가 두형이 헛소리에 무방비로 빵 터지는 컷 — 화당 1번 정도 들어가면 케미가 폭발
- **금지**: 은주를 \"맨날 화난 아내\" 로 그리지 말 것. 시큰둥은 사랑의 한 표현.

### [[이재윤|밤톨]] · [[이채은|곶감]] — \"카오스 / 동시성\"
- **싱크로 비트**: 둘이 동시에 같은 행동 (동시에 울음, 동시에 옹알이, 동시에 발차기). 동시성이 코믹의 핵심
- **반전 비트**: 한 명이 시작하면 다른 한 명이 따라하거나, 정반대로 가거나
- **무심한 빌런**: 부모가 진지한 순간에 둘이 발 잡고 옹알이 — \"이 와중에…\" 의 정서
- **L4 적용**: 부모가 영혼가출하는 동안 쌍둥이는 천진한 L1. 이 격차가 웃김.

### 트리오 케미 (두형+은주+쌍둥이)
- **두형 깐죽 → 은주 팩폭 → 쌍둥이 무심** 의 3단 비트가 한 화에 한 번 정도 들어가면 시리즈 정체성이 명확해짐
- 펀치라인 컷은 보통 \"두형 영혼가출\" 또는 \"은주 빵 터짐\" 에서 나옴

## 📋 화당 코믹 체크리스트

새 화 콘티를 짠 뒤 반드시 확인:

- [ ] **L4 또는 L5 표정 컷이 최소 1개** 들어갔는가?
- [ ] **펀치라인 컷 1~2개** 가 있는가? (단독 컷 + close/extreme-close + 만화적 이펙트)
- [ ] 펀치라인 앞 컷에 **spacerAfter: \"beat\" 또는 \"silence\"** 가 붙어 비트 쉼이 있는가?
- [ ] **컷 수가 6컷** 인가? → 8~10컷으로 늘릴 수 없는지 재검토 (대부분 늘릴 수 있음)
- [ ] **한 컷에 dialogue 3줄 이상** 있는가? → 분할 검토
- [ ] **샷 종류가 3가지 이상** 섞였는가? (medium만 6연속 금지)
- [ ] **두형이 깐죽 / 은주 팩폭 / 쌍둥이 카오스** 중 최소 2종이 들어갔는가?
- [ ] **만화적 이펙트** (땀줄기·핏줄·점선동공·입ㅁ자 등) 가 펀치라인에 적어도 1종 들어갔는가?
- [ ] **잔잔 톤 컷의 비율이 50% 이상** 인가? (코믹 남발도 금지, 격차로 웃기는 것)

## 🔄 적용 우선순위

1. **새로 작성하는 모든 storyboard** 는 이 룰을 디폴트로 따른다.
2. **기존 화 리메이크** 는 작가 요청 시에만 (4화부터 시범).
3. **이미 burn-in 된 이미지** 는 데이터(panels) 부터 고치고, 재생성은 별건.
