# 2화 · 엄마의 7시간

## 시놉시스
[[1화-한주의-낙|1화]]의 **직전 7시간** — 두형이가 사무실에서 "금요일 길다…" 하고 있을 때, [[오은주|은주]]는 집에서 쌍둥이 [[이재윤|밤톨이]]·[[이채은|곶감이]]랑 전쟁 중이었다. 든든한 아군은 오전 11시에 오시는 [[관리사님]] 한 분. 그리고 저녁 7시, 문 열리는 소리가 들린다.

## 컨셉
- **관점**: [[오은주]] 1인칭 — 나레이션이 은주 속마음.
- **톤**: 웃프게 7 : 따뜻하게 3.
- **분량**: **6컷** (1화와 동일 호흡, 인스타 정사각형).
- **1화와의 구조적 쌍**: 1컷(은주 시계) ↔ 1화 1컷(두형이 시계) / 6컷(은주 POV 현관) ↔ 1화 2컷(두형이 POV 현관). 같은 순간, 다른 시점.

## 6컷 흐름
1. **10:58 · 도입** — 은주 다크서클. 벽시계 2분 남음. (아이보리, 피로)
2. **11:00 · 관리사님 출근** — 문 열림. 구원의 벨소리. (크림, 따뜻)
3. **13:02 · 엄마 점심** — 편의점 김밥 한 입 베어문 채 곶감이 울음. (회색 유머)
4. **14:00 · 기저귀 대참사** — 동시 기저귀 카오스. 5컷 대칭의 카오스 버전. (크림, 카오스)
5. **15:00 · 1초의 평화** — 아이스아메리카노 꿀꺽. 두 어른의 공동 실눈. (황금빛, 정적)
6. **19:00 · 도착** — 문 열림. 은주 POV로 본 두형이. 1화 2컷의 쌍. (노란빛 전환)

## 모티프
- **시계**: 1·3·4컷에 시간 노출 (10:58 / 13:02 / 14:00). 시간이 안 간다는 체감 시각화.
- **은주 실눈 3단계**: 1컷(피곤) → 3컷(절망) → 6컷(복합 = 피곤+안도).
- **대칭 구도**: 4컷에서 대칭 기저귀 카오스. 1화엔 없던 "둘을 동시에" 구도의 시그니처.
- **문 모티프**: 2컷(관리사님 들어옴)과 6컷(두형이 들어옴) 수미상관.
