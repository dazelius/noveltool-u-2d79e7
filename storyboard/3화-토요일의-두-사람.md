# 3화 · 토요일의 두 사람

## 시놉시스
평일엔 [[관리사님]] 이 오전 11시에 오셨다. 평일엔 어쨌든 **어른 셋**이 쌍둥이 둘을 돌렸다.
그런데 오늘은 **토요일**. 관리사님이 안 오신다. 어른은 두 명 — **[[이두형|두형]]** 과 **[[오은주|은주]]**.

숫자는 딱 맞다. 1:1. 이론적으론.
그런데 왜 자꾸 **한 명이 더 필요한 것 같지?**

## 컨셉
- **관점**: 두 사람 공동 POV. 나레이션이 번갈아 등장. 같은 프레임 안에서 속마음이 **동시에** 뜨는 컷도 있음.
- **톤**: 웃프게 8 : 따뜻하게 2. 2화보다 살짝 더 웃프게, 1화보다 훨씬 솔직하게.
- **분량**: **6컷** (1·2화와 동일 호흡, 인스타 정사각형).
- **삼부작 마무리**: 1화(두형 단독 저녁) + 2화(은주 단독 낮) + 3화(둘 같이 하루) — **같은 주말의 세 조각** 구조.

## 6컷 흐름
1. **08:00 · 기상** — 침대 위 부감. 두 사람 같은 자세로 죽은 듯 누워있는데 아기 침대에서 울음. "오늘 토요일이지…" (연회색, 공포 도입)
2. **10:30 · 분업의 평온** — 두형이는 곶감이 기저귀, 은주는 밤톨이 수유. 둘 다 얼굴에 미소. 1:1 매치가 되고 있다. (크림톤, 잠깐의 평온)
3. **12:00 · 동시 호출 대참사** — 두 사람이 한 명씩 안고 있는데, **범퍼 의자에 혼자 있던 곶감이까지** 울기 시작. 팔이 모자람. (크림+카오스)
4. **15:00 · 좀비 구간** — 소파에 녹은 두 사람. 바닥은 장난감·쪽쪽이·빨래 폭격. 쌍둥이는 신나게 놀고 있음. **두 어른만** 방전. (회색유머)
5. **16:00 · 유모차 탈출** — 공원 벤치. 유모차 앞에서 둘 다 아이스아메리카노 쥐고 멍. 쌍둥이는 유모차에서 조용 — 드디어. (따뜻 포인트 ①)
6. **19:00 · 공동 실눈** — 쌍둥이 재우고 나온 거실. 맥주 꺼낼 기력조차 없어서 둘이 소파에서 마주본다. 서로 얼굴 보고 실눈 웃음 터짐. "해냈다." (따뜻 포인트 ②, 1·2화 엔딩의 쌍)

## 1·2화와의 구조적 연결
- **시계 모티프**: 1화(퇴근 19:00) + 2화(10:58 / 13:02 / 14:00 / 15:00 / 19:00) → 3화는 **08:00 / 10:30 / 12:00 / 15:00 / 16:00 / 19:00** 토요일 하루 풀타임.
- **문 모티프**: 1·2화 엔딩은 문이 열리면서 누군가 들어옴. 3화는 **문이 한 번도 안 열림** — 아무도 안 온다. 이게 주말의 진수.
- **실눈 모티프**: 1화(두형 반달눈) + 2화(은주 3단계 실눈) → 3화는 **둘이 동시에 실눈** 웃음. 공동 실눈의 탄생.
- **맥주 모티프**: 1화 맥주 축배 + 3화 맥주 못 꺼냄. "그 맥주가 금요일에만 허용되는 이유" 의 증명.

## 모티프
- **팔 수치**: 어른 2명 = 팔 4개. 쌍둥이 2명 = 잡혀줘야 할 손 4개. 이론상 맞지만 **한쪽이 동시에 두 가지 요구** 하면 바로 -1. 3컷에서 이 수치 파괴가 시각화.
- **공동 좀비**: 4컷. 1·2화에선 한 명씩 힘들었는데, 3화는 **둘이 나란히** 힘들다. 이게 이 화의 핵심 시각.
- **유모차 비밀병기**: 5컷. 주말 육아의 공식. 바깥=조용, 집=카오스.
